# Testimony Transcriber (Flutter Desktop)

Flutter desktop transcription client for court audio. Windows-first and Linux-compatible.

## Quick start

1. Ensure Flutter desktop is enabled for Windows/Linux.
2. Update `assets/config.json` with the correct API base URL if needed.
3. Run `flutter pub get`.
4. Launch: `flutter run -d windows` or `flutter run -d linux`.
